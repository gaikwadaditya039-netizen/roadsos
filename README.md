# RoadSOS 🚨

RoadSOS is a smart emergency assistance system developed to help people during road accidents and emergencies.

## Features
- Emergency SOS button
- Accident reporting
- Live location support
- Nearby hospital assistance
- User-friendly interface
- Fast emergency response system

## Technologies Used
- Python
- CustomTkinter
- SQLite / JSON
- APIs

## Future Improvements
- AI accident detection
- Voice emergency alert
- Real-time GPS tracking
- Ambulance booking
- Dark mode UI

## Project Goal
The goal of RoadSOS is to provide quick emergency assistance and improve road safety using technology.
